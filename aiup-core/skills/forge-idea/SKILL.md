# Forge Idea

## Overview
An interactive interview skill that pressure-tests your raw product concept to find holes, simplify the MVP, and ultimately produce a hardened `docs/vision.md` for AIUP.

## When to Use
- When you have a rough idea but aren't sure of all the details.
- When you want an agent to challenge your assumptions before you start building.

## Process
1. **The Interview**: Ask the user to describe their idea. 
2. **The Pressure Test**: 
   - Ask probing questions about edge cases, target audience needs, and the "why" behind the product.
   - Challenge complex features: "Do we really need X for the MVP?"
3. **Synthesis**: Once the idea is solid and agreed upon, compile the refined concept into a structured product vision.
4. **Save**: Output the final agreed-upon vision to `docs/vision.md`.
5. **Handoff**: Instruct the user to run `/requirements` to continue the AIUP process.
